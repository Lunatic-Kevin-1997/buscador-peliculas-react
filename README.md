# 🎬 Buscador de Películas - React App

![React](https://img.shields.io/badge/react-%2320232a.svg?style=for-the-badge&logo=react&logoColor=%2361DAFB)
![Vite](https://img.shields.io/badge/vite-%23646CFF.svg?style=for-the-badge&logo=vite&logoColor=white)
![API](https://img.shields.io/badge/API-RESTful-success?style=for-the-badge&logo=json)

Una aplicación web interactiva construida con **React** que permite a los usuarios buscar películas en tiempo real, conectándose a una API externa para obtener títulos, años de lanzamiento y pósters oficiales.

🔗 **[Mira la aplicación en vivo aquí](https://buscador-peliculas-react-smoky.vercel.app/)**
---

## ✨ Características Principales

- **Búsqueda Dinámica:** Los usuarios pueden ingresar el título de cualquier película y obtener resultados instantáneos.
- **Consumo de API:** Integración con una API externa (RESTful) utilizando peticiones asíncronas (`fetch`) para traer los datos de las películas.
- **Renderizado de Listas:** Uso eficiente del método `.map()` en React para generar una cuadrícula (grid) dinámica con las tarjetas de cada película.
- **Manejo de Estados y Errores:** Interfaz amigable que muestra mensajes claros cuando no se encuentran resultados o si el campo de búsqueda está vacío.
- **Diseño Responsivo:** Galería de películas que se adapta perfectamente a pantallas de celulares, tablets y monitores de escritorio.

---

## 🛠️ Tecnologías Utilizadas

- **Frontend:** React 18 (Hooks: `useState`, `useEffect`)
- **Arquitectura:** Consumo de APIs REST / JSON
- **Herramienta de Construcción:** Vite
- **Estilos:** CSS
